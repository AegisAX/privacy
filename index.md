## 1. 요약

K-REFCON은 사용자가 지정한 ThingSpeak 채널의 냉동 컨테이너(리퍼) 센서 데이터를 실시간으로
표시하고, 온도 이탈·센서 이상 시 경보하는 **모니터링 도구**입니다.

- 이 앱은 **광고를 게재하지 않으며, 분석(analytics)·추적(tracking)·크래시 리포팅을 사용하지 않습니다.**
- 이 앱은 **자체 서버로 어떠한 개인정보도 수집·전송하지 않습니다.** 제공자(㈜케이리퍼콘)는
  사용자의 데이터를 수집하지 않습니다.
- 사용자가 입력·설정한 모든 정보는 **기기 내부에만 저장**됩니다.

## 2. 앱이 처리하는 정보

앱 기능을 위해 다음 정보가 **기기 내부에 로컬로** 저장됩니다.

| 항목 | 저장 위치 | 용도 |
|---|---|---|
| ThingSpeak 채널 ID | 기기 저장소(SharedPreferences) | 모니터링 대상 채널 식별 |
| ThingSpeak Read API 키 | 기기 보안 저장소(iOS Keychain / Android Keystore, 암호화) | 비공개 채널 데이터 조회 |
| AI 제공자 API 키 / OAuth 토큰(선택) | 기기 보안 저장소(암호화) | 선택적 AI 분석 기능 |
| 알람 임계값·필드 설정·표시 설정 | 기기 저장소 | 사용자 설정 유지 |
| 조회된 센서 값·알람 이력(최근 200건) | 기기 저장소 | 화면·위젯 표시, 사후 확인 |

이 앱은 이름·이메일·위치·연락처·기기 식별자 등 **개인 식별 정보를 수집하지 않습니다.**

## 3. 외부로의 데이터 전송

앱은 다음의 경우에만, **사용자의 요청·설정에 따라** 외부 서비스와 통신합니다. 어느 경우에도
제공자(㈜케이리퍼콘)의 서버로는 전송되지 않습니다.

- **ThingSpeak** (`api.thingspeak.com`): 사용자가 등록한 채널의 센서 데이터를 조회합니다.
  사용자가 입력한 Read 키가 함께 전송됩니다(HTTPS).
- **AI 분석 제공자**(사용자가 "AI 분석"을 실행할 때만): 선택한 제공자(Google Gemini /
  Anthropic Claude / OpenAI)의 API로 **다운샘플된 센서 통계·시계열**을 전송해 분석 결과를
  받습니다. 전송되는 데이터는 온도 등 센서 수치이며 개인정보는 포함하지 않습니다. 해당 데이터는
  전송 시점 각 제공자의 개인정보처리방침·이용약관의 적용을 받습니다.

모든 네트워크 통신은 **HTTPS(전송 중 암호화)** 로 이루어집니다.

## 4. 권한 사용 목적

| 권한 | 목적 |
|---|---|
| 인터넷 / 네트워크 상태 | 센서 데이터 조회 |
| 알림(POST_NOTIFICATIONS) | 온도 이탈·센서 이상 경보 표시 |
| 전체 화면 인텐트(USE_FULL_SCREEN_INTENT) | 긴급 경보 시 잠금/절전 화면을 깨워 알림 |
| 포그라운드 서비스(FOREGROUND_SERVICE_DATA_SYNC) | 앱을 닫아도 주기적으로 폴링해 상시 경보 |
| WAKE_LOCK / RECEIVE_BOOT_COMPLETED | 백그라운드 경보의 신뢰성 유지 |
| 배터리 최적화 예외(선택) | 백그라운드 경보 지연·차단 방지(사용자 동의 기반) |
| 진동(VIBRATE) | 경보 시 진동 |

## 5. 데이터 보관 및 삭제

- 모든 데이터는 **기기 내부에만** 보관되며 제공자에게 전송·보관되지 않습니다.
- 사용자는 앱 내에서 채널·API 키·설정을 **언제든 삭제**할 수 있습니다.
- **앱을 삭제(제거)하면 앱이 저장한 모든 데이터가 함께 삭제**됩니다.
- 앱은 `allowBackup=false`로 설정되어 있어, 저장한 정보가 클라우드 자동 백업에 포함되지 않습니다.
